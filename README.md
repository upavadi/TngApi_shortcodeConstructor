# Shortcodes are

- `TngWp_birthdays` List birthdays for the selected month.
- `TngWp_danniversaries` List death anniversaries for the selected month.
- `TngWp_gone` Gone But Not Forgotten. 
- `TngWp_LandingPage`
- `TngWp_manniversaries`
- `TngWp_MyShortcode`
- `TngWp_submitImage`
